# Bash script template

To initiale a bash script.


Bash short option only.

By default the color is enabled.  
You can disable this by setting this variable before running the script :  
export NO_COLOR=1  
./my_template.sh ....  
  

Reset to default :
unset NO_COLOR  
./my_template.sh ....
